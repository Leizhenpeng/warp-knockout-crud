## 介绍

不要紧张，这只是一个简单的CRUD，但是它可以帮助你理解如何使用Warp构建后端restful风格的API。

## Feature

- [x] 用Rust编写
- [x] 使用Warp构建
- [x] 基于Tokio的异步
- [x] 日志记录

## develop

```bash
# first clone the repo
git clone git@github.com:Leizhenpeng/warp-knockout-crud.git
# then cd into the repo,
cd warp-knockout-crud
# install the dependencies
make install

# run the project
make dev
```
